How to make script.

- "text" - 텍스트 출력
- "clear" - 텍스트 비우기
- "bg" - 배경 출력
- "char" - 캐릭터 출력
- "pos" - (char과 함께 사용) 캐릭터 위치 설정 , center/right/left 사용가능  [default : center] 
- "audio" - 오디오 출력
- "stopAudio" - 오디오 정지
- "loops" - (audio와 함꼐 사용) 반복 횟수 설정 , 반복변수 : -1 [default : 1]
- "shake" - 화면 흔들림, 1부터 100까지 설정가능
