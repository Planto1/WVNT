How to make script.

- "bg" - 배경 출력
- "char" - 캐릭터 출력
- "audio" - 오디오 출력
- "loop" - (오디오와 함꼐 사용) 반복 횟수 설정, 미설정시 자동으로 1번 출력, 무한 반복은 -1
- "shake" - 화면 흔들림, 1부터 100까지 설정가능
